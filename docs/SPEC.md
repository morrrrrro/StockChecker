# プロダクト仕様書

## 概要

東証上場銘柄を対象とした株価日次レポートアプリケーション。
毎日の市場動向を分析し、定量的スクリーニングに基づいた「買い目」の銘柄を提示する。

## 目的とゴール

- 日次の市場動向を5分以内で把握できるレポートを自動生成する
- マルチファクタースコアリングにより、感覚ではなくデータに基づいた銘柄選定を行う
- リスク指標を常に併記し、偏りのない判断材料を提供する

## 設計思想

### ノイズ排除

金融ニュースの99%はノイズ（Bloomberg Intelligence）。
情報過多は意思決定の精度を下げるため、このアプリは「行動につながるシグナル」のみを提示する。

- 5年後に影響しない情報は載せない
- アナリスト目標株価の追従的変更は除外
- 軽い出来高での小幅な変動は無視

### リスクの常時表示

買いシグナルには必ずリスク指標を併記する。
機会だけを見せて、リスクを隠すレポートにはしない。

### 行動バイアス対策

- アンカリング防止: 52週高値ではなくバリュエーション指標をリードに
- フレーミング対策: 「高値から-15%」と「安値から+40%」の両参照点を提示
- 損失回避/処分効果対策: 価格変動ではなく投資テーゼの変化を重視する「テーゼチェック」セクション

## 対象市場・銘柄

- **市場**: 東京証券取引所（プライム・スタンダード・グロース）
- **銘柄リスト**: JPX公式の上場銘柄CSVから取得
- **初期対象**: 日経225構成銘柄を中心に、段階的に拡大

## レポート構成

レポートは7つのセクションで構成される。
**後ろ振り返り30% : 前向きアクション70%** のバランスを維持する。

### 1. 市場環境サマリー（30秒で把握）

前日の海外市場・為替・債券の動向を一覧する。

| 指標 | 重要度 | 理由 |
|---|---|---|
| 前日NY（Dow/S&P500/Nasdaq） | 必須 | 日本市場寄付きの最強予測因子 |
| 日経225先物（CME/大阪夜間） | 必須 | 当日の日経予想値 |
| USD/JPY（20日トレンド付き） | 必須 | 日経との相関 ≈ 0.81（20日ベース） |
| 米10年債利回り | 必須 | グロース株バリュエーションに直結 |
| VIX | 必須 | リスクセンチメント |
| TOPIX | 必須 | 日経の大型株偏重を補完 |
| WTI原油 | 参考 | エネルギーセクター・投入コスト |

### 2. 今日のポイント（2-3行）

「何が起きたか」ではなく「だから今日どうなるか」を記述する。
最大3行。ナラティブの要約ではなく、行動に影響する情報のみ。

### 3. セクター動向ヒートマップ

東証33業種のセクター動向をTreemapで可視化する。

- サイズ: 時価総額
- 色: 日次リターン（赤=上昇、青=下落、日本市場慣習）
- セクターローテーションシグナル（景気局面との対応）を付記

### 4. スクリーニング結果

3つのスクリーニング戦略の結果を提示する（詳細はSCREENING.md参照）。

- Screen A: バリュー＋クオリティ（Greenblatt-Piotroski ハイブリッド）
- Screen B: モメンタムブレイクアウト
- Screen C: 配当バリュー
- **収束シグナル**: 複数スクリーンに同時出現する銘柄を強調

### 5. ウォッチリスト状況

監視中の銘柄・保有銘柄の状況を表示する。

各銘柄に表示する情報:
- 株価・変動率・出来高比率
- 5日間スパークライン
- シグナルバッジ（新規/強化中/安定/減衰中）とライフサイクル日数
- 確信度（1-5段階）
- サポート/レジスタンス水準とリスク/リワード比率
- ATR・β値・F-Score

### 6. 今週のカタリスト

- 決算発表カレンダー（予想EPS付き）
- 経済指標リリース予定（予想値 vs 前回値）
- 日銀会合、SQ日、配当権利落ち日

### 7. テーゼチェック

保有銘柄で「投資理由を変えるべき変化」があったかだけを表示する。

- 変化なし → 何も表示しない（ノイズ排除）
- 変化あり → 何が変わったか、投資テーゼへの影響を簡潔に記述

## 更新頻度

- **日次バッチ**: 毎営業日18:00 JST（東証終了後、データ確定を待って実行）
- **スケジューラ**: GitHub Actions（月〜金、cronスケジュール）
- **データ蓄積**: Parquetファイルをgitリポジトリにcommitして永続化

## 閲覧方法

| 方法 | 用途 | アクセス |
|---|---|---|
| **GitHub Pages** | 日次レポート閲覧（メイン） | ブラウザからURL（PC・スマホ対応） |
| **Streamlit** | インタラクティブな深掘り分析 | ローカルで `git pull` → `streamlit run` |

GitHub Pagesには毎日自動的に静的HTMLレポートがデプロイされる。
PCを起動していなくてもレポートは生成・公開される。
